The app will keep track of the eating places and alert the user if he has been eating to much fast food recently.
  • Use Core Location framework to get user’s location every 5 minutes
  • Use Google Places API to retrieve nearby places, set types=food to retrieve restaurant, use parameter types=food and keyword="fast food restaurant" to retrieve fast food restaurants
  • If the distance from the user to the place is less than a certain limit(10m), the user is considered as being in the restaurant.
